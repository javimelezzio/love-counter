# javimelezzio – love-counter

Small HTML/CSS/JS project:
- Envelope intro → letter → heart screen
- Floating hearts + click explosion
- Live “time since” counter

## Run locally
Just open `index.html` in a browser (or use VS Code Live Server).

## Customize
- Text: edit `index.html`
- Styles/background: `styles.css`
- Phrases + start date: `script.js` (`frases` array + `start` date in `actualizarContador()`)
